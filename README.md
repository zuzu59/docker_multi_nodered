# docker_multi_nodered
Petits tests pour juste lancer 10x containers NodeRed sur 10x ports différents
zf211019.1207

# Démarrer
Juste faire:

```
./start.sh
```

Les instances seront alors:

http://your_ip:18081/

http://your_ip:18082/

http://your_ip:18083/

...


# Arrêter
Juste faire:

```
./stop.sh
```
